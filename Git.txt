git update-git-for-windows
更新Windows上的Git

git init
在当前目录下创建Git版本库

git add 文件名
将文件添加到暂存区（版本库）

git add .
将文件夹内所有文件添加到暂存区

git commit -m "Note"
提交暂存区文件到本地库并备注Note

一般每次都要用到git add和git commit -m ""


git status
查看当前版本库内文件

git log
查看提交记录

git checkout -b dev
创建名为dev的分支
